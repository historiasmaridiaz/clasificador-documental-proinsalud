Publicar el Clasificador Documental

Esta guía se puede abrir con doble clic en Windows.

No suba solamente este HTML. El clasificador necesita Python para OCR, TRD/CCD, SQLite, PDF, Excel y ZIP. Debe cargar todos los archivos y carpetas del proyecto.

1. Crear repositorio

Abra github.com/new, cree un repositorio privado y use un nombre como clasificador-documental-proinsalud.

2. Cargar el proyecto

Descomprima el ZIP, entre en clasificador_documental_proinsalud y cargue todo su contenido. En la raíz de GitHub deben verse app.py, requirements.txt, packages.txt, la carpeta core y la carpeta data.

3. Convertirlo en web

Abra share.streamlit.io, seleccione el repositorio, la rama main y escriba app.py como archivo principal.

4. Contraseña opcional

En Secrets agregue:

APP_PASSWORD = "SU_CONTRASEÑA_SEGURA"

Privacidad: para documentos institucionales use repositorio privado y no publique historias clínicas o datos personales en una aplicación abierta.
